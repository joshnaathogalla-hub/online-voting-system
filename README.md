 Online Voting System

## Project Description
A full-stack online voting system where users can securely vote and admin can view results.

---

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

---

##  Features
- User Registration & Login
- Unique Voter ID verification
- One vote per user
- Admin login for results
- Real-time vote counting

---

## How to Run

### Backend
1. Open in Eclipse
2. Run Spring Boot App

### Frontend
1. Open in VS Code
2. Run:
   npm install
   npm start

---

## API Endpoints

- POST /api/auth/register  
- POST /api/auth/login  
- POST /api/vote/cast  
- GET /api/vote/results
